# Simple URL Shortener

A **minimal full-stack URL shortener** built with **FastAPI**, **SQLite**, and **HTML/CSS**.  
Users can enter a long URL and get a short, easy-to-share link. Clicking the short link redirects to the original URL.

---

## Features

- Shorten long URLs into short codes
- Redirect short URLs to the original URL
- Simple and clean UI with inline CSS
- Lightweight and easy to run
- Uses SQLite for persistent storage

---

## Tech Stack

- **Backend:** Python, FastAPI
- **Database:** SQLite
- **Frontend:** HTML, CSS (inline), minimal JS
